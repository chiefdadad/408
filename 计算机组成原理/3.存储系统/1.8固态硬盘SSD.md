# 固态硬盘SSD

1. ### 固态硬盘的结构：

   ![image-20250628194948781](images/image-20250628194948781.png)

   **注（易混淆）：**

   - 固态硬盘是以页为一个读写单位的，因此SSD中系统要读/写的逻辑块号指的是一个页而不是块；

   - SSD中的页对应的是磁盘存储器中的扇区，而块对应的是磁盘中的一个磁道。
