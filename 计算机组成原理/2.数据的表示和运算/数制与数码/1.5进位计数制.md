# r进制计数法

基数：每个数码位所用到的不同符号个数，r进制的基数为r。

2. ### 二进制 <->八进制、十六进制：

   - **二进制 -> 八进制：**

     每3位一组，转化为对应的八进制符号。

   - **二进制 -> 十六进制：**

     每4位一组，转化为对应的十六进制符号。

   - **八进制 -> 二进制：**

     每一位八进制转换为3位对应的二进制。

   - **十六进制 -> 二进制：**

     每一位十六进制转换位4位二进制。
