# 算数逻辑单元ALU

​		一种组合逻辑电路，实现了**加 / 减 / 乘 / 除、与 / 或 / 非** 等功能，**ALU是运算器的核心**（CPU由**控制器**和**运算器**构成）。

​		由于加减乘除等运算都要基于“加法”实现，因此**加法器是ALU的核心**。
